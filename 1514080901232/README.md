## 小程序创意说明
这是一个简单的作业通知的小程序

## 小程序页面设计
  首页：小程序主页面（最新通知显示在首页中）  
  写通知页面：写新的通知，有字数限制。 
  历史通知页面：可查看，可重新编辑，可删除通知的页面。

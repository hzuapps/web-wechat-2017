## 小程序创意说明
这是一个简单的记录笔记的小程序

## 小程序页面设计
  笔记页面：小程序主页面（可以查看当前自己的笔记）  
  添加笔记页面：书写新的笔记  
  编辑笔记页面：用来查看或删除笔记（请求、保存数据）  
  反馈页面：一个简单的反馈界面（表单界面）

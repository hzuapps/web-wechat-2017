## 创意说明：
这个小程序主要是为了让想购买手机的人事先对手机有个大概的了解，当然也只是做给想了解这方面的人看的，不感兴趣的其实看了也是没多大意思的，当然
做得有点仓促，没有构思好，如果有机会，之后会做个更加好的出来！供购买手机的人参考，也是一个建议！
## 页面设计：
首页就一个搜索功能，可以搜索小程序里面想要的信息！

第二页也就是一个集齐手机的基本信息，根据这些信息来让选购手机的人做出判断！

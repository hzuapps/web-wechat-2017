# WechatApp-TodoList

一个基于微信小程序的任务清单。包括所有任务列表、未完成列表和已完成列表，可以标识为未完成和已完成。

## Demo

![](https://github.com/aisin/WechatApp-TodoList/blob/master/doc/screenshot.gif?raw=true)

## 使用

1. 克隆此项目到本地

2. 在微信开发工具中创建项目

3. 选择该项目目录

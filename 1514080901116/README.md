## 创意说明：
Umore，幽默的谐音，是一个带给人快乐的小程序。主要页面有三个，分别是首页、分享和登录界面。

## 页面设计：
首页目前还没有完善，主要有一个图库和一个删除缓存的功能。分享页面既是一个可以滚动的界面（带有循环数组），也是一个可以评论的界面。登录界面带有一个反馈页面，也就是表单，可以提交显示。
